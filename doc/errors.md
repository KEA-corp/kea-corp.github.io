# ERREUR
## Erreurs externe

- **php command not found...**\
[↗ installer PHP](https://www.php.net/downloads.php)

- **Could not open input file: inter.php**
```shell
cd <chemin du projet>
```
- **Erreur: pas de fichier**\
Le fichier kea n'existe pas/est introuvable.
```shell
php inter.php <nom du fichier kea>
```
